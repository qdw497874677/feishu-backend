问题：长连接接收的消息 content 字段是 JSON 字符串格式 {"text":"xxx"}，直接传递给 SDK 导致报错 'content is not a string in json format'。解决方案：在 MessageListenerGatewayImpl.convertToMessage 方法中解析 JSON 内容，提取 text 字段的纯文本。代码：使用 Gson 解析，当 content 以 '{' 开头时，解析为 JsonObject，获取 text 字段。效果：消息发送成功，机器人能正常回复。注意：飞书长连接和 WebHook 模式接收的消息格式都是 JSON 字符串。
